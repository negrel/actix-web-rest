# `actix-web-rest` - Opiniated toolkit to build restful API using actix-web.

## Why ?

:zap: Gotta go fast!

## Getting started

```shell
# Create a new project.
cargo init my_api

# Add actix-web-rest to your project.
cargo add
````

### Building the project

```shell
make build
```

### Running the tests

```shell
make tests
```
